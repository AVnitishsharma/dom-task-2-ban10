# 👽 Ban 10 Aliens

A fan-made **Ben 10 Alien Encyclopedia** built using **HTML, CSS, and JavaScript**. Explore different aliens, their species, home worlds, and other details through a futuristic Omnitrix-inspired interface.

> **⚠️ Disclaimer:** This is a fan project created for educational and portfolio purposes only. Ben 10 and its characters are owned by their respective copyright holders.
